## Step-by-Step Setup

### 1. Install Dependencies
Open your terminal in the taskmate directory and run:
```bash
npm install
```

This will install all required packages:
- React & React DOM
- React Router DOM
- Framer Motion (for animations)
- React Icons & Lucide React
- Vite (build tool)

### 2. Start Development Server
```bash
npm run dev
```

The app will open automatically in your browser at `http://localhost:3000`

### 3. Explore the Application

#### Public Pages (No Login Required)
1. **Home (/)** - Landing page with hero section
2. **Features (/features)** - Feature details
3. **Pricing (/pricing)** - Pricing plans
4. **About (/about)** - About the company
5. **Blog (/blog)** - Blog posts
6. **Contact (/contact)** - Contact form

#### Try the App
7. Click **"Sign up"** or **"Log in"** button
8. On the login page, click **"ðŸš€ Try Demo Account"**
9. You'll be redirected to the Dashboard!

#### App Pages (After Login)
- **Dashboard (/app/dashboard)** - Overview with stats
- **Projects (/app/projects)** - Manage projects
- **Tasks (/app/tasks)** - Task management
- **Team (/app/team)** - Team members
- **Settings (/app/settings)** - Account settings
